What is CSS ?

  1. CSS Stands for Cascading Style Sheet.
  2. CSS describes how html elements are to be displayed on screen, paper, or in other media.
  3. CSS is not neccessary CSS saves a lot of work. It can control the layout of multiple web pages all at ones
  4. external stylesheets are stored in css files.

css can be added to html Document in 3 ways:

  • Inline- by using the style attribute inside HTML elements1
  • internal- by using a style element in the head section
  • external- by using a link element to link to an external css files
  • the most common to add css, is to keep css styles in external CSS files. However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate , and easier for you to try it yourself.

when comparing css vs ID, the difference is that class applies a style to multiple elements. ID on the other hand, applies a style to one unique element. ID is also special in that you can use a special URl to link directly to an element and it's used by javascript

in CSS, sectors are used to target a specific element or range of the element on a webpage. once an element has been targeted, a style or set of styles can be applied to the element.
